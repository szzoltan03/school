Beadandó leírás:

Beadandó esettanulmány követelményei
A gyakorlati jegy megszerzéséhez egy esettanulmányt, vagyis egy komplett
adatelemzési munkát kell elvégezni és dokumentálni. Az adatelemzés technikai részét
Python Notebookban kell kivitelezni, de a vizualizációkat, az eredmények bemutatását
lehet BI programokkal (Power BI, Tableau) csinálni.
Ez történhet az internetről letöltött adatokkal, de lehet saját adatforrással is. A
komplett elemzés a következőkből álljon:
Egy legalább 5 oldalas beszámoló az elemzésről. Készítsük úgy, mintha leendő
főnökünket akarnánk lenyűgözni
A dokumentációt a következő szempontok
alapján készítse el:
• Mi a célja az elemzésnek?
• Milyen adatforrásokat használ az elemzéshez, ezeket hogyan tisztította meg?
• Milyen módszereket használt, ezeknek mik matematikai, statisztikai alapjai?
• Milyen eredményekre, következtetésekre jutott az elemzés során? Ezeket, ha
kell vizuálisan mutassa be.
• Milyen javaslatai vannak (pl. a megrendelőnek) az eredmények alapján?
Kitérhet a továbbfejlesztési lehetőségekre is.
• Csatolni kell az adatokat, forrásprogramot (ipynb, py).
• Csatolni lehet az eredmények bemutatását segítő, táblázatokat, diagramokat
és műszerfalakat. (Power BI, Tableau fájlok)
• Nem kötelező, de sokat dobna az elemzésen, ha egy videóban foglalná össze
a készítő a feladat megoldását.
• Érdemes a témaválasztásnál a jövő félévi szakdolgozatban is gondolkodni.
