# ENG
# Data Analysis Assignment

This repository contains a university assignment completed within the Business Data Analytics specialization.

## Project Objective

The goal of this assignment was to analyze a dataset, perform data preprocessing, and apply basic analytical methods using Python.

## Files

- `beadando.md` – Assignment requirements
- `beadando.docx` – Written report
- `beadando.ipynb` – Main analysis notebook
- `oraifeladatok.ipynb` – Independent class exercises used for learning purposes only (not part of the assignment)

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
